### object 타입이란?

정의 : object는 모든 타입의 기본 타입으로, 모든 데이터 타입을 담을 수 있음
(typescript에서의 unknown과 비슷?)

### C#의 enum이란?

정의 : enum은 열거형 타입으로, 상수 값들의 집합을 정의할 때 사용
(정해져 있는 것들을 enum으로 정의해서 사용하면 좋음(예: 요일, 계절 등))

```csharp
enum Days
{
    Sunday,
    Monday,
    Tuesday,
    Wednesday,
    Thursday,
    Friday,
    Saturday
}
```
